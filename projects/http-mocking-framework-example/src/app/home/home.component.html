An easy-to-use framwork that provides Mocking strategies for developing HTTP-based components in your Angular projects.

TODO Sample Application

The TODO Sample Application demonstrates how to create a TODO list app using a business layer based on DAOs.

It shows how to use the HTTP Mocking Framework to facilitate the development of REST-based apps, built on top of an abstraction layer.

If you are not familiar with Data Access Object architecture, please refer to the basic HTTP Mock Service Demo instead.
