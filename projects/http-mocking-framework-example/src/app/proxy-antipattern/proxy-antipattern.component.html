Instructions

Please follow the instructions below to make this example work:

  • You must have a PHP server installed on your local machine.
  • Create a project directory called angular-toolbox into your server.
  • Copy the folder api/message and its content from the php directory to the angular-toolbox directory on your server.
  • Create a virtual host in yout PHP server with the host name angular-toolbox.
  • Start the PHP server.
