AbstractVersionManager Class

Since v 0.9.3

The abstract class for all services that implement the VersionManager interface.

Description

The scope of the VersionService class is set to "root". This means that all functionality of this service is shared between all components of your application.

You may want to implement a version management service with another scope (e.g. to create a component library). You can do that by simply extending the AbstractVersionManager class.

Signature

The AbstractVersionManager class implements the VersionManager interface.
See the VersionManager interface API for details. 

abstract class AbstractVersionManager implements VersionManager
