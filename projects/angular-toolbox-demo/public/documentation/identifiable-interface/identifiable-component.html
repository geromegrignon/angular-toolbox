IdentifiableComponent Class

Identifiable objects are components that can be identified by a unique Uuid instance identifier.

constructor

constructor(classRef: string | undefined = undefined)

Creates a new IdentifiableComponent instance.

Parameters
classRef: string | undefined

A string that represent the class reference for this object. This can be useful to workaround TypeScript compilation obfuscation.
When the value is undefined, the class reference is defined by the value of the constructor name property.
Default value is undefined.

IdentifiableComponent.getId()

getId(): Uuid

Gets this unique identifier of this object.

Returns
Uuid

A Uuid instance that represents the unique identifier of this object.
See Uuid class for details.

IdentifiableComponent.getClassRef()
