Identifiable Interface

Identifiable objects are components that can be identified by a unique Uuid instance identifier.

Identifiable.getId()

getId(): Uuid

Gets this unique identifier of this object.

Returns
Uuid

A Uuid instance that represents the unique identifier of this object.
See Uuid class for details.

Identifiable.getClassRef()

Since v 0.9.2 
getClassRef(): string

Returns the class name reference of this object.
This is useful to workaround TypeScript compilation obfuscation.

Returns
string

The class name reference of this object.
See IdentifiableComponent class for details.
