Dark Mode Service

A lightweight service that provides easy-to-use Dark Mode implementation for your Angular application.

API

Interface Description Link
DarkModeConfig Defines properties used to configure DarkModeService instances.
Property Description Link
change The callback event that is triggered when the dark state has changed.
Method Description Link
toggleDarkMode() Toogles the dark mode state.
disableDarkMode() Sets the dark mode state to inactive.
enableDarkMode() Sets the dark mode state to active.
darkModeEnabled() Indicates whether the dark mode state is active, or not.
getStorageKey() Gets the value of the storage key as defined by the config provider.
getCssProperty() Gets the value of the CSS property as defined by the config provider.
invalidateStorage() Removes the dark mode settings information from local storage.
Constant Description
STORAGE_KEY Reference to the dark mode default storage key. Value is "dark-mode-key"
CSS_PROP Reference to the dark mode default css property. Value is "dark-mode"
Provider Description Link
DARK_MODE_CONFIG The default provider for the DarkModeService configuration.
