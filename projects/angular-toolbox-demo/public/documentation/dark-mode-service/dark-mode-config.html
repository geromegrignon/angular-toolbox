DarkModeConfig Interface

The DarkModeConfig defines properties used to configure DarkModeService instances.

DarkModeConfig.darkModeEnabled

darkModeEnabled?: boolean

A boolean value that indicates whether the dark mode is activated by default (true), or not (false).

DarkModeConfig.detectBrowserSettings

detectBrowserSettings?: boolean

A boolean value that indicates whether the dark mode uses browser settings (true), or not (false).
When true, this property overrides the darkModeEnabled property.

DarkModeConfig.cssProperty

cssProperty?: string

The name of the CSS property used to set the dark mode look and feel.

DarkModeConfig.storageKey

storageKey?: string

The reference to the key value used to persist the dark mode state to local storage.
