NavigateToUrlDirective

An easy-to-use directive that enables keyboard navigation and provides support for navigating to an external URL.

The NavigateToUrlDirective class makes the decorated element enable for keyboard navigation and accessibility (ARIA).

API

Directive Description Link
navigateToUrl Navigates to the external URL specified by the href attribute of the decorated element.
Attribute Description
href The href attribute of the decorated element.
