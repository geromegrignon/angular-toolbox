ArrayList Class

Since v 0.9.5

The ArrayList class uses a backing Array as the source of the data collection. Items in the backing Array can be accessed and manipulated using the methods and properties of the ArrayList instance. Operations on an ArrayList instance modify the data source.

Motivation

API

Event Description Link
ArrayListEvent The ArrayListEvent class represents an event that is dispatched when the associated collection changes.
Property Description Link
change Dispatches events each time the list is modified.
length Gets the number of items in the list.
Method Description Link
addAll() Adds a list of items to the current ArrayList instance.
addItem() Adds the specified item to the end of the list.
addItemAt() Adds the item at the specified index.
getItemAt() Gets the item at the specified index.
getItemIndex() Returns the index of the item if it is in the list.
removeAll() Removes all items from the list.
removeItem() Removes the specified item from this list.
removeItemAt() Removes the item at the specified index and returns it.
toArray() Returns an array that is populated in the same order as this ArrayList instance.
