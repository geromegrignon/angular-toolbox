ArrayList.addItemAt() Method

The ArrayList.addItemAt() method adds an item to the list at the specified index.

Signature

addItemAt(item: T, index: number): ArrayListy<T>
Parameters
item: <T>

The item to place at the index.

index: number

The index at which to place the item.

Returns

The reference to the ArrayList instance.

Events
ArrayListEvent

An ArrayListEvent of type of ArrayListEventType.ADD.

Exeptions
RangeError

Thrown if index is less than 0 or greater than the length of the list.
