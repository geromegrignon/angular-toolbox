ArrayList.removeItemAt() Method

The ArrayList.removeItemAt() method removes the item at the specified index and returns it. Any items that were after this index are now one index earlier.

Signature

removeItemAt(index: number): T | undefined
Parameters
item: T

The index from which to remove the item.

Returns

The item that was removed.

Events
ArrayListEvent

An ArrayListEvent of type of ArrayListEventType.REMOVE.
