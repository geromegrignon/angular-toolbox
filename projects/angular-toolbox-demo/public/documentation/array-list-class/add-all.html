ArrayList.addAll() Method

The ArrayList.addAll() method adds a list of items to the current ArrayList instance.

Signature

addAll(addList: Array<T>): ArrayListy<T>
Parameters
addList: Array<T>

The list of items to add to this current ArrayList instance.

Returns

The reference to the ArrayList instance.

Events
ArrayListEvent

An ArrayListEvent of type of ArrayListEventType.ADD_ALL.
