ArrayListEvent Event

The ArrayListEvent class represents an event that is dispatched when the associated collection changes.

Event Types

Event Type Description
ArrayListEventType.ADD_ALL The type of event associated with the ArrayList.addAll() action.
ArrayListEventType.ADD The type of event associated with the ArrayList.addItem(), or ArrayList.addItemAt() actions.
ArrayListEventType.REMOVE_ALL The type of event associated with the ArrayList.removeAll() action.
ArrayListEventType.REMOVE The type of event associated with the ArrayList.remove(), or ArrayList.removeAt() actions.

ArrayListEvent.type

readonly type: ArrayListEventType

The type of this event.

ArrayListEvent.list

readonly list: ArrayList<T>

The reference to the ArrayList instance that dispatches this event.
