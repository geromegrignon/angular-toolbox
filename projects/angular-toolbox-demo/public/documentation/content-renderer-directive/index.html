ContentRendererDirective

The ContentRendererDirective class allows injection of any value into an HTML element container (div, button, etc.).

Motivation

API

Directive Description Link
contentRenderer A directive declaration that provides dynamic injection capabilities to an HTML element.
Attribute Description Link
content The value to be rendered within the decorated HTML element.
Event Description Link
rendered Dispatches events when the content of the decorated HTML element has been rendered.
