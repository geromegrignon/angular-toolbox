Global Constants

A set of global constant definitions that are regularly used in Angular applications.

Constant Description
EMPTY_STRING A reference to an immutable empty string: "".
UNDEFINED A reference the "undefined" JavaScript type.
OBJECT A reference the "object" JavaScript type.
BOOLEAN A reference the "boolean" JavaScript type.
NUMBER A reference the "number" JavaScript type.
BIGINT A reference the "bigint" JavaScript type.
FUNCTION A reference the "function" JavaScript type.
STRING A reference the "string" JavaScript type.
SYMBOL A reference the "symbol" JavaScript type.
