Scroll Service

A lightweight service that provides scrolling capabilities to your Angular application.

Description

API

Property Description Link
onScroll The callback function that is triggered when a scroll operation is performed.
Method Description Link
scrollIntoView() Scrolls the ancestor containers of an HTML element with the specified selector, such that this element is visible to the user.
gotoTop() Scrolls the app viewport to the top of the client window.
scroll() Scrolls the app viewport according to the specified options, into the client window.
scrollBy() Scrolls the app viewport by the specified [x, y] coordinates, into the browser window.
