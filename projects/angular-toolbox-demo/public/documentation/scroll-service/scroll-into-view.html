ScrollService.scrollIntoView() Method

The ScrollService.scrollIntoView() method allows to scroll the ancestor containers of an HTML element with the specified selector, such that this element is visible to the user.

Signature

scrollIntoView(selector: string, arg?: boolean | ScrollIntoViewOptions | undefined): void
Parameters
selector: string

The selector of the HTML target element.

arg?: boolean | ScrollIntoViewOptions | undefined

Can be either a boolean value, a ScrollIntoViewOptions object, or undefined.

If a boolean value:

  • if true, the top of the element will be aligned to the top of the visible area of the scrollable ancestor.
  • if true, the bottom of the element will be aligned to the bottom of the visible area of the scrollable ancestor.
Otherwise, a ScrollIntoViewOptions object is dictionary containing the scroll parameters. See element.scrollIntoView on the MDN website for details.

Default value is undefined.
