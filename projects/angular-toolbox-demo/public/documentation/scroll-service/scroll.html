ScrollService.scroll() Method

The ScrollService.scroll() method allows to scroll the app viewport according to the specified options parameter, into the client window.

Signature

scroll(options?: ScrollToOptions): void
Parameters
options: ScrollToOptions

An optional diationary with the following properties:

  • left?: number
  • top?: number
  • behavior?: ScrollBehavior
See element.scroll on the MDN website for details.
