ScrollService.gotoTop() Method

The ScrollService.gotoTop() method allows to scroll the app viewport to the top of the client window.

Signature

gotoTop(behavior: ScrollBehavior = 'smooth'): void
Parameters
behavior: ScrollBehavior

Specifies whether the scrolling should animate smoothly (smooth), happen instantly in a single jump (instant), or let the client choose (auto, default).
Default value is "smooth".
