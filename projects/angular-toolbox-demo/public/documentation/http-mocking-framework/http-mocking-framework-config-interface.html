HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockingFrameworkConfig Interface

Since v 0.11.3

The HttpMockingFrameworkConfig interface defines the API for objects that are used to set the HTTP Mocking framework configuration.

HttpMockingFrameworkConfig.disableVisualFlag

disableVisualFlag?: boolean

An optional boolean value that indicates whether the visual flag is visible (false), or not (true).
The visual flag is visible when this property is not defined.

HttpMockingFrameworkConfig.productionPolicy

productionPolicy?: HttpMockProductionPolicy

An optional value that indicates the behavior of the framework when it is used in production mode.
The framework uses the HttpMockProductionPolicy.ERROR value when this property is not defined.
See the HttpMockProductionPolicy enum reference for details.
