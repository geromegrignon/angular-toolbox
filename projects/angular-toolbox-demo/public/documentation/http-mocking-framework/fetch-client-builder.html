HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

FetchClientBuilder Class

Since v 0.11.1

A static utility class for building FetchClient objects.
You tipically use FetchClient objects as parameter of the HttpResponseMockBuilder.body() method.

FetchClientBuilder.buildFetchClient()

static buildFetchClient(input: RequestInfo | URL, init: RequestInit | null = null, responseType: FetchClientResponseType = FetchClientResponseType.JSON): FetchClient

Builds and returns a new FetchClient object.

Parameters
input: RequestInfo | URL

The definition of the resource to fetch. This can either be:

  • A string or any other object with a stringifier — including a URL object — that provides the URL of the resource you want to fetch.
  • A Request object.
    See the Request API reference on MDN for details.
init: RequestInit | null = null

The object containing options to configure the request.
Default value is null.

See the RequestInit API reference on MDN for details.

responseType: FetchClientResponseType = FetchClientResponseType.JSON

A constant of the FetchClientResponseType that sepcifies the type of response for the resource to fetch.
Default value is FetchClientResponseType.JSON.

See the FetchClientResponseType API reference for details.

Returns
FetchClient

A new FetchClient object.
