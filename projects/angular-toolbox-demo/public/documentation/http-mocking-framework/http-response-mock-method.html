HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

httpResponseMock Function

A utility function used to easily create new chainable HttpResponseMockBuilder instances.

Signature

()=> HttpResponseMockBuilder
Returns
HttpResponseMockBuilder

A new instance of the HttpResponseMockBuilder class.

Usage

You typically use the httpResponseMock function in a mock congig to create new HttpResponseMockBuilder instances through the Builder Method Design Pattern.

Method Chaining:

Method chaining is used to invoke multiple methods on the same object which occurs as a single statement.
