HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HTTP_MOCKING_FRAMEWORK_CONFIG Provider

Since v 0.11.3

The HTTP_MOCKING_FRAMEWORK_CONFIG allows to setup the configuration of the HTTP Mocking Framework.

disableVisualFlag

disableVisualFlag: false

Indicates whether the visual flag is visible (false), or not (true).
Default value is false.

productionPolicy

productionPolicy: HttpMockProductionPolicy.ERRO

Indicates the behavior of the framework when it is used in production mode.
Default value is HttpMockProductionPolicy.ERROR.
See the HttpMockProductionPolicy enum reference for details.
