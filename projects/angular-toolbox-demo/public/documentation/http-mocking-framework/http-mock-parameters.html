HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockParameters Interface

The markup interface that defines the type of parameter objects associated with a HTTP call. It represents objects passed as second parameter of the HttpMethodMock.data() method.

HttpMockParameters are vanilla JavaScript objects that contain information extracted from the route associated with an HTTP request.

For example, the route /users/:id/:age will produce a HttpMockParameters object as shown bellow: 

{ id: <value>, age: <value> }
And in a real world example: 
this.httpClient.get("/users/10/33").subscribe();
// Parameters: { id: 10, age: 33 }
