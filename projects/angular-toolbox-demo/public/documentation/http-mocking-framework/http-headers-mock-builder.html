HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpHeadersMockBuilder Class

A statefull builder for creating new HttpHeaders instances.
See the HttpHeaders reference for more details.

HttpHeadersMockBuilder.cacheControl()

cacheControl(value: string = "no-cache"): HttpHeadersMockBuilder

Sets the "Cache-Control" property of the new HttpHeaders instance.

Parameters
value: string

The value used to set the "Cache-Control" HTTP header value.
Set tos "no-cache" by default.

Returns
HttpHeadersMockBuilder
