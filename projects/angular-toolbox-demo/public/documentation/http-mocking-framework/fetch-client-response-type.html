HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

FetchClientResponseType Enum

Since v 0.11.1

Defines the possible values for the responseType property of the FetchClientBuilder.buildFetchClient() method.

Value Description
ARRAY_BUFFER Allows to return the response as a promise that resolves with an ArrayBuffer.
BLOB Allows to return the response as a promise that resolves with a Blob.
FORM_DATA Allows to return the response as a promise that resolves with a FormData object.
JSON Allows to return the response as a promise which resolves with the result of parsing the body text as JSON.
TEXT Allows to return the response as a promise that resolves with a String.
The response is always decoded using UTF-8.
RESPONSE A special type that allows to return the response as a promise that resolves with the original Response object.
