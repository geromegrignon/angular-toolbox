HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockProductionPolicy Enum

Since v 0.11.3

Defines the possible values for the productionPolicy property of the HttpMockingFrameworkConfig interface.

Value Description
SILENT Indicates that the framework intercepts requests silently in producion mode.
WARNING Indicates that the framework has to send a warning message to the console when a request is intercetped in producion mode.
ERROR Indicates that the framework trhows an error when a request is intercetped in producion mode.
