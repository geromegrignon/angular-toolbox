HTTP Mocking Framework

The HTTP Mocking Framework provides an easy-to-use and highly configurable solution to mock HTTP responses to API calls sent from your Angular applications.

HttpMockError Interface

An interface that provides the API that must be implemented by HTTP mock errors.

HttpMockError.status

status: HttpStatusCode

The status of the HTTP error.

HttpMockError.statusText

statusText: string

The status text of the HTTP error.
