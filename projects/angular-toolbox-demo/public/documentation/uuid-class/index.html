Uuid Class

The Uuid class is used to parse, construct and manage UUIDs in the context of an Angular app.

This class is not available in workers.

Motivation

A Universally Unique Identifier (UUID) is a 128-bit label used for information in computer systems.

The Uuid class provides a safe encapsulation mechanism for working with UUID strings. UUIDs encapsulation should be considered by developers as a part of OOP design brest practices.

For example, using instances of the Uuid class guarantees that your application won't corrupt UUID strings managed by server side.

API

Instance Method Description Link

Class Method Description Link

fromString() Creates and returns a new Uuid instance built from the specified string parameter.
validate() Performs validation on the specified UUID string parameter.
isTracked() Performs a test to check whether a UUID string is tracked by the system, or not.
