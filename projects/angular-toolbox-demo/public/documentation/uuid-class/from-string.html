Uuid.fromString() Static Method

The Uuid.fromString() static method creates a new Uuid instance built from the specified UUID string parameter.

Signature

static fromString(uuid: string, track: boolean = false): Uuid
Parameters
uuid: string

The UUID string value used to create the new Uuid instance.

track: boolean

If true, makes the associated UUID string elligible for tracking by the system.

Default value is false.

Returns

A new Uuid instance.

Exeptions
IntegrityError

Thrown if the track parameter is true and the string specified by the uuid parameter is already tracked by the system.

TypeError

Thrown whether the specified uuid parameter is not valid UUID string.
