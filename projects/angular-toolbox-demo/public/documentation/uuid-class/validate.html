Uuid.validate() Static Method

The Uuid.validate() static method Performs validation on the specified UUID string parameter.

Signature

static validate(uuid: string): boolean
Parameters
uuid: string

The UUID string on which to perform validation.

Returns
boolean

true whether the uuid string parameter is valid; false otherwise.
