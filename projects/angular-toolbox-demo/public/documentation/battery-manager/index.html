BatteryManager Interface

Since v 0.10.1

The BatteryManager interface of the Battery Status API provides information about the system battery charge level. The navigator.getBattery() method returns a promise that resolves with a BatteryManager interface.

BatteryManager.charging

readonly charging: boolean

A boolean value indicating whether the battery is currently being charged.

BatteryManager.chargingTime

readonly chargingTime: number

A number representing the remaining time in seconds until the battery is fully charged, or 0 if the battery is already fully charged.

BatteryManager.dischargingTime

readonly effectiveType: number

A number representing the remaining time in seconds until the battery is completely discharged and the system suspends.

BatteryManager.level

readonly level: number

A number representing the system battery charge level scaled to a value between 0.0 and 1.0.

BatteryManager.onchargingchange

onchargingchange(event: Event): void

An event fired when the battery charging state (the charging property) is updated.

Parameters
event: Event

A generic Event.

Use the event name in methods like addEventListener(), or set an event handler property.
