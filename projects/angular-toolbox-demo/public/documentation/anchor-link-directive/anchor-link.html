anchorLink Directive Declaration

The anchorLink allows to scroll to a DOM element and to update the URL fragment with the href attribute of the decorated element.

Description

The anchorLink directive declaration provides the following functionalities to HTML elements:

  • adds a tabindex attribute to the decorated element (set to 0)
  • adds a role attribute to the decorated element (set to link)
