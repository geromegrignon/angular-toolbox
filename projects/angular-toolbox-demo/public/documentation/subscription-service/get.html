SubscriptionService.get() Method

The SubscriptionService.get() method all Subscription instances associated with the specified reference.

Signature

get(ref: string | Identifiable): Array<Subscription> | null
Parameters
ref: string | Identifiable

The reference for which to get all Subscription instances.
See Identifiable for details.

Returns
Array<Subscription> | null

All Subscription instances associated with the specified reference, or null whether the specified reference does not exists.

Exeptions
TypeError

Thrown when the ref parameter is neither a string, nor an AbstractIdentifiable instance.

Usage
