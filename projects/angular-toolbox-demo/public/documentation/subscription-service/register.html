SubscriptionService.register() Method

The SubscriptionService.register() method allows to strore new Subscription instance, associated with the specified reference.

The Subscription will be unsubscribed when the SubscriptionService.clearAll() method will be invoked for the associated reference.

Signature

register(ref: string | Identifiable, subscription: Subscription): SubscriptionService
Parameters
ref: string | Identifiable

The reference for which to store a new Subscription instance. Can be either a string or an Identifiable object.
See Identifiable for details.

subscription: Subscription

The Subscription instance to register.
See Subscription for details.

Returns
SubscriptionService

A reference to the SubscriptionService instance.

Exeptions
TypeError

Thrown when the ref parameter is neither a string, nor an IdentifiableComponent instance.

Usage
