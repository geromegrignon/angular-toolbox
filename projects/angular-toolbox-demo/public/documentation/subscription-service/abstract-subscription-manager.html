AbstractSubscriptionManager Class

Since v 0.9.3

The abstract class for all services that implement the SubscriptionManager interface.

Description

The scope of the SubscriptionService class is set to "root". This means that all functionality of this service is shared between all components of your application.

You may want to implement a subscription management service with another scope (e.g. to create a component library). You can do that by simply extending the AbstractSubscriptionManager class.

Signature

The AbstractSubscriptionManager class implements the SubscriptionManager interface.
See the SubscriptionManager interface API for details. 

abstract class AbstractSubscriptionManager implements SubscriptionManager
