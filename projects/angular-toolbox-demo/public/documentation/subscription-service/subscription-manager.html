SubscriptionManager Interface

Since v 0.9.3

The SubscriptionManagerinterface defines the API that must be implemented by Subscription Managment Services.

API

Method Description Link
register() Stores a new Subscription instance associated with the specified object, or reference.
append() Stores a new Subscription instance, associated with the reference specified by the last invokation of the register() method.
clearAll() Unsubscribes and removes all Subscription instances associated with the specified reference, or object.
get() Returns all Subscription instances associated with the specified reference, or object.
