SubscriptionService.append() Method

The SubscriptionService.append() method allows to register a new Subscription instance associated with the reference specified by the last register() method invokation.

The Subscription will be unsubscribed when the SubscriptionService.clearAll() method will be invoked for the associated reference.

Signature

append(subscription: Subscription): SubscriptionService
Parameters
subscription: Subscription

The Subscription instance to register.
See Subscription for details.

Returns

A reference to the SubscriptionService instance.

Exeptions
SubscriptionError

Thrown when the register() method have not been called before this append() method invokation.
