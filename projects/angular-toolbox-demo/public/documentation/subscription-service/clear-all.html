SubscriptionService.clearAll() Method

The SubscriptionService.register() allows to strore new Subscription instance, associated with the specified reference. The Subscription will be unsubscribed when the SubscriptionService.clearAll() method will be invoked for the associated reference.

Signature

clearAll(ref: string | Identifiable): boolean
Parameters
ref: string | Identifiable

The reference for which to unsubscribe and remove all Subscription instance. Can be either a string or an Identifiable object.
See Identifiable for details.

Returns

true whether the specified reference exists and all Subscription have been removed; false otherwise.

Exeptions
TypeError

Thrown when the ref parameter is neither a string, nor an AbstractIdentifiable instance.

Usage
