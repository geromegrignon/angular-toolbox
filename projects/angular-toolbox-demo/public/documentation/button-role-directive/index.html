ButtonRole Directive

An easy-to-use directive that enables keyboard navigation and provides support for keyboard "Enter" key events.

The ButtonRole directive makes the decorated element enable for keyboard navigation.

Motivation

In software development, keyboard navigation is a must have. The concept of SaaS and the use of platforms like Electron make keyboard navigation an essential functionality for building quality applications.

The ButtonRole directive aims to implement such functionality by removing tedious coding tasks. Additionally, it provides built-in ARIA elements to support accessibility.

API

Directive Description Link
buttonRole Enables keyboard navigation and provides support for keyboard Enter key events.
Attribute Description Link
delegateClick Forces callback methods associated with the enter event taregt to be invoked when User clicks on the decorated element.
Event Description Link
enter Dispatches events when the user presses the Enter key.
