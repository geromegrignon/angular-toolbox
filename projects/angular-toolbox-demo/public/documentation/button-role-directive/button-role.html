buttonRole Directive Declaration

The buttonRole directive declaration enables keyboard navigation and provides support for keyboard Enter key events.

Description

The buttonRole directive declaration provides the following functionalities to HTML elements:

  • provides direct biding to keyboard Enter events
  • adds a tabindex attribute to the decorated element (set to 0)
  • allows to bind a click event with the Enter event declaration
  • adds a role attribute to the decorated element (set to button)
