Angular Toolbox Quick Start Guide

Angular Toolbox is a library of lightweight utilities gathered to solve recurring issues encountered by Angular apps developers.

The library focuses on ease-of-use and robust test processes, to maintain quality over time.

Prerequisites

The Angular Toolbox library is built on top of the Angular framework. So, make sure that you have created an Angular project with the minimum required version before starting the Angular Toolbox install: 

minimum angular version: 18.0.3

Peer Dependencies

The Angular Toolbox library needs the following Angular modules to be installed to work: 

"@angular/common"
"@angular/core"
"@angular/platform-browser"
"@angular/router"

Install

Angular Toolbox is available for download at npm. 

$cd <YOUR-PROJECT-NAME>
$npm install angular-toolbox

Usage

Each utility can be imported individually so that you only bundle what you use. 

import { Component } from '@angular/core';
import { DarkModeService } from 'angular-toolbox';

@Component({
    selector: 'app-custom-component',
    standalone: true,
    templateUrl: './custom.component.html'
})
export class CustomComponent {
    
    constructor(darkModeService: DarkModeService) {}
}

Running Tests

Angular Toolbox unit testing is the process of testing all pieces of code provided with the library.

Run the the unit testing suite with the npm test command: 

$npm test
