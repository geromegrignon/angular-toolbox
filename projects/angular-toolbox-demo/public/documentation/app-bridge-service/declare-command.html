AppBridge.declareCommand() Method

The AppBridge.declareCommand() allows to register a new JavaScript commands to be used with dynamically loaded HTML documents.

Signature

declareCommand(name: string, command: AppBridgeCommand): void
Parameters
name: string

The name of the JavaScript command to declare.

command: AppBridgeCommand

The reference to the JavaScript command to declare.
See AppBridgeCommand Interface

Usage
